# e-plantShopping

## Paradise Nursery - Online Plant Shop

A React-based shopping cart application for an online plant shop offering various house plants.

### Features
- Landing page with navigation
- Product listing with categorized plants
- Shopping cart with quantity management
- Dynamic cart updates
- Responsive design

### Project Structure
- `App.jsx` - Main application component
- `ProductList.jsx` - Product listing page
- `CartItem.jsx` - Individual cart item component
- `CartSlice.jsx` - Redux slice for cart management
- `AboutUs.jsx` - About page component

### Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
